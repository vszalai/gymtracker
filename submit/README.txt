Npm was used as a package manager.
Dependencies for the frontend: 
@tailwindcss/vite
axios
pinia
socket.io-client
tailwindcss
vue
vue-router

Dependencies for the backend:
cookie-parser
cors
debug
dotenv
express
mongodb
mongosh
morgan
socket.io

"npm run dev" to run the frontend
"npm run start" to run the backend


The frontend can be found in the "gymtracker" folder. The default ports should be 5173 and 3000 for the front and backend respectively.
Sample .env for the backend (it should be placed in backend/)
"
MONGO_USER=a
MONGO_PW=
MONGO_CLUSTER=
"
